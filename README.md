# Jukes World — starter site

Static starter website for a parent brand with two branches:

- `/music/`
- `/art/`

## Files

- `index.html` — parent brand homepage
- `music/index.html` — music branch
- `art/index.html` — art branch
- `about/index.html` — shared brand story page
- `contact/index.html` — shared contact page
- `styles.css` — shared styles
- `script.js` — mobile nav and placeholder form behavior
- `CNAME` — custom domain for GitHub Pages

## Setup for GitHub Pages

1. Create a GitHub repository.
2. Upload all files and folders in this project to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set the source to deploy from your main branch.
5. Confirm the custom domain uses the `CNAME` file value.
6. Point your DNS records for `jukesworld.com` to GitHub Pages if needed.
